# Changelog

All notable changes to the Abstract Claimer project will be documented in this file.

## [2.4.0] - 2026-01-20
### Added
- Added support for new Genesis allocation tiers.
- Integrated improved Gas Estimator to reduce transaction costs by 30%.
- Added support for proxy lists in `config.json`.

### Fixed
- Fixed connection timeout errors on Windows 11.
- Resolved issue with "Tier 2" wallet verification.

## [2.1.0] - 2025-12-15
### Changed
- Updated RPC endpoints for Abstract Mainnet.
- Improved anti-sybil detection bypass.

## [1.5.0] - 2025-11-05
### Added
- Initial support for Pudgy Penguins holders.
- Added multi-threading support.

## [1.0.0] - 2025-10-20
- Initial release.
